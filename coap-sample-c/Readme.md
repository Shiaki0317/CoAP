# Readme

libcoap を利用するときの説明を行う，

## libcoap

以下が libcoap のホームページとなる．
このリンク先より libcoap の導入手順やリファレンス、RFC の説明ページなどに飛ぶことができる．

- [libcoap](https://libcoap.net/)

## リファレンス (libcoap)

libcoap 向けのリファレンスは以下のリンクを参照する．
このとき、利用するバージョンに対応したリファレンスを参照すること．
また，バージョン指定では URL"<https://libcoap.net/doc/reference/4.3.5/">にて設定されるバージョン"4.3.5"を利用するライブラリのバージョンに変更する．

- [libcoap リファレンス (develop)](https://libcoap.net/doc/reference/develop/)
- [libcoap リファレンス (v4.3.5)](https://libcoap.net/doc/reference/4.3.5/)

## テストサーバ

CoAP テスト用に利用するサーバは[coap.me](https://coap.me/)となる．

ここで利用できるメソッドや URI は以下のリンクを参照して設定する．

- [ETSI test driver](https://coap.me/test/coap://coap.me)
- [well-known](https://coap.me/coap://coap.me/.well-known/core)
